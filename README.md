# 🌦️ Weather App (JavaScript)

## 📌 Description
A simple and clean weather application that fetches real-time weather data for any city using an API.

This project was originally created earlier and has been recently refactored to improve structure, UI, and code quality.

---

## 🚀 Features
- 🔍 Search weather by city name
- 🌡️ Displays temperature (current, min, max)
- 💧 Shows humidity level
- 🌥️ Displays weather condition
- ⚡ Real-time API integration
- 🎨 Clean and modern UI

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (Vanilla JS)
- OpenWeatherMap API

## ▶️ How to Run Locally
1. Clone the repository
2. Open `index.html` in your browser
3. Enter city name and search
---
## 🔑 API Setup
This project uses OpenWeatherMap API.

1. Get your free API key from: https://openweathermap.org/
2. Replace in `app.js`:

javascript
`:const apiKey = "YOUR_API_KEY";`
